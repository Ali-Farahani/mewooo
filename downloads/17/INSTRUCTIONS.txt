Downloaded YouTube files are in this folder.

If the video was split, you will see files like:
  video.mp4.part-000
  video.mp4.part-001
  video.mp4.part-002

To reassemble split files on Linux:
  cat *.part-* > output.mp4

Example:
  cat *.mp4.part-* > output.mp4

If there is a normal .mp4 file and no .part-* files, just download the .mp4 file directly.
